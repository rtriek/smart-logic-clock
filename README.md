# 🕒 Smart Productivity Clock with BI Integration

A modern, high-performance digital clock built with **JavaScript** and **Glassmorphism UI**. This tool doesn't just show time; it generates actionable data for **Business Intelligence (Power BI)** analysis.

---

## 🚀 Overview
This project is designed for professionals who want to track their focus sessions and analyze their productivity patterns. It bridges the gap between **Web Development** and **Data Analytics**.

### Key Features:
* **Real-time Tracking:** Accurate digital clock with localized English formatting.
* **Data Generation:** Logs "Study" and "Break" activities with precise timestamps.
* **BI-Ready Export:** Download your activity logs as a `.csv` file directly compatible with **Power BI** or **Excel**.
* **Modern UI:** Sleek Glassmorphism design with responsive visual feedback.
* **Data Control:** Integrated Reset functionality to manage data sessions.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Advanced Gradients & Backdrop Filters).
* **Logic:** Vanilla JavaScript (ES6+).
* **Data Format:** CSV (Comma-Separated Values).
* **Analysis Tool:** Power BI Desktop (Recommended).

---

## 📊 How to use for Business Intelligence?
1.  **Record:** Use the "Start Study" and "Take a Break" buttons during your work session.
2.  **Export:** Click "Download Data for Power BI" to get your `productivity_log.csv`.
3.  **Analyze:** * Open **Power BI Desktop**.
    * Select `Get Data` -> `Text/CSV`.
    * Create visualizations (Pie charts for activity ratios, Line charts for time distribution).
